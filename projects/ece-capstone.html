3D Object Reconstruction

Electrical and Computer Engineering Capstone Project

As part of our Electrical and Computer Engineering degree program, we completed a group Capstone project that needed use signal processing and/or pattern recognition in its solution. Our project used point capture technology to obtain depth points for real world objects, and then process them to create 3D mesh models inside a virtual environment. This project used Nearest Neighbor Filtering, Multiple Viewpoint Registration, Euclidean Clustering for Segmentation, and Fast Triangulation Meshing.
